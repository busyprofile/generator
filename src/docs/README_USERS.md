# Инструкция по созданию пользователей

## Автоматическое создание пользователей

Для автоматического создания пользователей с разными ролями выполните команду:

```bash
php artisan app:create-users
```

Эта команда создаст следующих пользователей:

1. **Администратор** (Super Admin)
   - Email: admin@example.com
   - Пароль: password

2. **Трейдер** (Trader)
   - Email: trader@example.com
   - Пароль: password

3. **Мерчант** (Merchant)
   - Email: merchant@example.com
   - Пароль: password

4. **Тимлидер** (Team Leader) - создается только если роль существует в системе
   - Email: teamleader@example.com
   - Пароль: password

## Создание пользователей через админ-панель

Для создания пользователей через админ-панель:

1. Войдите в систему как администратор
2. Перейдите в раздел "Пользователи"
3. Нажмите кнопку "Создать пользователя"
4. Заполните форму:
   - Имя - имя пользователя
   - Почта - email пользователя (должен быть уникальным)
   - Пароль - пароль пользователя
   - Подтверждение пароля - повторите пароль
   - Роль - выберите роль пользователя (Super Admin, Trader, Merchant, Team Leader)
5. Нажмите кнопку "Сохранить"

После создания пользователя автоматически создается кошелек и назначается выбранная роль.

## Доступные роли в системе

В системе доступны следующие роли:

1. **Super Admin** - администратор системы с полным доступом
2. **Trader** - трейдер, который обрабатывает заказы
3. **Merchant** - мерчант, который создает заказы
4. **Team Leader** - руководитель команды трейдеров (если роль существует в системе)

## Создание пользователей через сидер

Также можно запустить сидер для создания пользователей:

```bash
php artisan db:seed --class=UserSeeder
```

или запустить все сидеры:

```bash
php artisan db:seed
```
